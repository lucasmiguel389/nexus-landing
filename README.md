# Nexus — Landing Page

Landing page para um produto SaaS fictício de automação de workflows com IA, criada como projeto de portfólio. HTML, CSS e JavaScript puros — sem frameworks, sem build step.

![preview](./preview.png)

## ✨ Sobre o projeto

O produto fictício, **Nexus**, conecta ferramentas (Stripe, Slack etc.) e usa IA para decidir a próxima ação de um workflow. A landing page foi construída em torno desse conceito, com:

- Hero com imagem gerada por IA e um diagrama de workflow animado em SVG, com pulso de luz percorrendo as conexões
- Seções de recursos, como funciona, planos e formulário de cadastro
- Paleta de cores própria: fundo azul-marinho quase preto, acento âmbar e "wire teal", fugindo do combo genérico preto + verde neon
- Tipografia: Space Grotesk (títulos), Inter (corpo) e JetBrains Mono (detalhes técnicos)

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (variáveis, Flexbox, Grid)
- JavaScript puro (animação via SVG e `requestAnimationFrame`, sem dependências)
- Google Fonts

## 🚀 Como rodar localmente

Não há dependências ou build. Basta clonar e abrir o arquivo:

```bash
git clone https://github.com/lucasmiguel389/nexus-landing.git
cd nexus-landing
```

Depois é só dar duplo clique no `index.html`, ou usar a extensão **Live Server** no VSCode.
